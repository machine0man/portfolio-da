Titanic Data Analysis

Dataset Overview

This dataset contains passenger-level data from the Titanic disaster.

Dataset Source: View on Kaggle

  • PassengerId: Unique identifier
  • Survived: 1 = survived, 0 = not
  • Pclass: Passenger class
  • Sex: Gender
  • Age: Age
  • SibSp / Parch: Family onboard
  • Fare: Ticket price
  • Embarked: Boarding port

Setup

import pandas as pd
import sqlite3

df = pd.read_csv("data/titanic-data.csv")

conn = sqlite3.connect("titanic.db")
df.to_sql("titanic", conn, if_exists="replace", index=False)

SQL Analysis

1. Total male and female passengers 
SELECT Sex, COUNT(*) 
FROM titanic 
GROUP BY Sex;
2. Overall survival rate
SELECT ROUND(AVG(Survived)*100,2) FROM titanic;
3. Survival by gender
SELECT Sex, COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY Sex;
4. Survival by class
SELECT Pclass, COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY Pclass;
5. Average fare per class
SELECT Pclass, ROUND(AVG(Fare),2)
FROM titanic GROUP BY Pclass;
6. Average age (survived vs not)
SELECT Survived, ROUND(AVG(Age),2)
FROM titanic GROUP BY Survived;
7. Survival by gender & class
SELECT Pclass, Sex, COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY Pclass, Sex;
8. Paid above avg fare & survived
SELECT * FROM titanic
WHERE Fare > (SELECT AVG(Fare) FROM titanic)
AND Survived=1;
9. Top 5 highest fares
SELECT * FROM titanic ORDER BY Fare DESC LIMIT 5;
12. Survival by fare range
SELECT CASE
WHEN Fare < 10 THEN '0-10'
WHEN Fare < 50 THEN '10-50'
WHEN Fare < 100 THEN '50-100'
ELSE '100+' END,
COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY 1;
13. Family size effect
SELECT (SibSp+Parch), COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY 1;
14. Alone vs family
SELECT CASE WHEN (SibSp+Parch)=0 THEN 'Alone'
ELSE 'Family' END,
COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY 1;
16. Family size buckets
SELECT CASE
WHEN (SibSp+Parch)=0 THEN 'Alone'
WHEN (SibSp+Parch)<=3 THEN 'Small'
ELSE 'Large' END,
COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY 1;
17. Survival by embarkation
SELECT Embarked, COUNT(*), SUM(Survived),
ROUND(AVG(Survived)*100,2)
FROM titanic GROUP BY Embarked;
18. Survived & above class avg fare
SELECT * FROM titanic
WHERE Survived=1 AND Fare>(
SELECT AVG(Fare) FROM titanic t2
WHERE t2.Pclass=titanic.Pclass)
LIMIT 10;
